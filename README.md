## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/Preety36/banana-harvesting-datasets.git
   cd banana-harvesting-datasets
   ```

2. Install the required library:
   ```bash
   pip install requests
   ```

3. Run the script:
   ```bash
   python download_datasets.py
   ```
